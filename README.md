# Evolution Diary App

Evolution Diary App is a web application for tracking daily habits and activities, helping users achieve their goals and maintain a healthy lifestyle.

## Table of Contents
- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)
- [Screenshots](#screenshots)
- [Contributing](#contributing)

## Features
- Track various habits and activities
- Set and achieve daily targets
- Dynamic progress tracking
- Interactive and user-friendly interface
- Fully responsive

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/dtsitovich/evolution-diary-app.git
   cd evolution-diary-app

## Usage
- Click on the habit buttons in the side menu to view and track your progress for each habit
- Add comments and records for each day to keep track of your achievements
- Click last button in the side menu to add new habit category(menu button)
